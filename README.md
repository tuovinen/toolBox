<h1>toolBox of Scripts</h1>

*This Github repository contains few selected scripts and programs I use to help my daily work as a service supervisor. All code is written by me.*

| Program  | Language(s) | Description |
|---|---|---|
| IP address_heroku  | Python | A tool to test HTTP(S) requests and/or get an IP address of the source host. |
| JSON_kaunistaja  | HTML, JS, CSS | Makes compact and minimized JSON easier for human to read. |
| NPS scripts  | VBA, Python, Powershell | Scripts to help get desired data from order emails. |
| Outlook helpers  | VBA | Small programs to help handling emails etc. |
| RPA_tuntikirjaus  | Python, Sikulix | A simple RPA-ish script made with Sikulix. |
| Ratti  | VBA | VBA script to separate sheets from an Excel workbook to their own files. |
